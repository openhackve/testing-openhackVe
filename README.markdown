# OpenHack

The community site for organizing OpenHack groups! Please read the [Your City](http://openhack.github.com/yours) to see how to start your own.

Get started!

    gem install bundler
    bundle

Run the site with:

    rake
### Adicional - VenSec

Para el site de Vensec, debes:

	gem install bundler
	bundle init
	echo "gem 'rails'" >> Gemfile
	bundle install

Y luego, el comando:

	rake
